HOW TO INSTALL A PACE NOTES LOG
-------------------------------

Copy the log file into the following directory:

<Your Documents folder>\TGD Navigator\Logs\Autosave\<Game Name>

PLEASE PAY ATTENTION!

Some tracks need a "maprecord" file, which is neeeded to make TGD Navigator works.
To install the maprecord file, copy it into the following directory:

C:\Program Files (x86)\SimHub\PluginsData\<Game Name>\MapRecords
